# Conexion fastApi con PHP

## Instalacion y configuracion
1. Instalar FastAPI
    ```bash
    ~$  pip install fastapi  
    ~$  pip install uvicorn  
    ```
2. encender el servicio de fastAPI con uvicorn
    ```bash
    ~$ uvicorn main:app --reload 
    ```
> Nota: http://127.0.0.1:8000/docs muestra informacion de la api
